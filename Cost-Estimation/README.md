## Cost Estimation

This architecture is highly cost-efficient due to its serverless design.

Estimated monthly cost for **100,000 API requests**:

| Service              | Estimated Cost |
| -------------------- | -------------- |
| API Gateway          | ~$3.50         |
| AWS Lambda           | ~$0.20         |
| DynamoDB (On-demand) | ~$1.00         |
| SNS Notifications    | ~$0.10         |
| CloudWatch Logs      | ~$0.50         |

### Estimated Total

~ **$5 – $6 per month**

Actual costs may vary depending on request volume and usage patterns.
